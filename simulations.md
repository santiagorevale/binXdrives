# Simulations

A detailed list of the commands used to run the simulations for the publication can be found below, classified in groups.


## To test performance (not part of the paper)
```bash
Rscript --vanilla binXdrives.R -t 8 -s 6 -g 24 -k 1800 -f 48 -p 3 -a 60 -b 60 --release_critical_difference 100 --bed_design yes -c 0.9 --resistance_formation 0.05 --dominance 0.6 -e 0 --intended_fecundity_cost 1 --unintended_reproductive_cost_A 0.025 --unintended_reproductive_cost_B 0.025 --unintended_viability_cost_A 0.025 --unintended_viability_cost_B 0.025 -o fertility_BED_reference
```
